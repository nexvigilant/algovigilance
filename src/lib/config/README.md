## Purpose

The Config module centralizes the design and structural configuration for content across the platform. It handles thematic styling for the Intelligence Hub, image generation presets (AI), and the sequential ordering of content series (curriculum logic).

---

## File Manifest

| File | Type | Purpose |
|------|------|---------|
| `intelligence-themes.ts` | Config | Maps content types to specific color palettes and UI themes |
| `image-style-presets.ts` | Config | Prompt modifiers for consistent AI-generated imagery |
| `series.ts` | Config | Defines the ordering and hierarchy of content series |
| `index.ts` | Barrel | Centralized exports for all configuration |
