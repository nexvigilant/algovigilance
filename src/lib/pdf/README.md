## Purpose

The PDF module handles the generation of clinical-grade documents, completion certificates, and data exports. It provides both low-level PDF primitives and high-level generators for specific platform artifacts.

## File Manifest

| File | Type | Purpose |
|------|------|---------|
| `certificate-pdf-generator.ts`| Service | High-precision certificate creation for course completion |
| `pdf-export.ts` | Service | General purpose export logic for reports and research artifacts |
| `index.ts` | Barrel | Central entry point for PDF generation |
